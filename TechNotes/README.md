# TechNotes - 个人技术笔记系统

一个用于管理技术笔记和学习资料的个人项目。

## 📋 项目结构

```
TechNotes/
├── docs/          # 技术文档
├── src/           # 源代码
└── config/        # 配置文件
```

## ✨ 功能特性

- 📝 支持 Markdown 格式笔记
- 🎨 自定义主题配置
- 🔄 自动保存功能

## 🚀 快速开始

```bash
# 克隆仓库
git clone <repository-url>

# 进入项目目录
cd TechNotes

# 查看帮助
python src/note_manager.py --help
```
